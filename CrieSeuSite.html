Crie seu primeiro website com o que aprendeu!


Depois de sua jornada de aprendizado no Conhecimento Dev ©, que tal criar o seu primeiro website com base no que aprendeu? Guiaremos os passos principais para a criação de um site simples, justificando algumas escolhas e apontando as modificações que o projeto fique com sua cara.

A maioria dos pontos não será expressa de forma muito detalhada, caso qualquer dúvida, retorne as páginas HTML e CSS do nosso site, e não se esqueça dos links que podem ampliar seus conhecimentos, localizados ao lado direito das abas do site.


  • 1° Passo:


Escreva uma exclamação "!" em seu editor de código para criar uma básica estrutura html, contendo suas principais tags inseridas. Ficará desta forma:


<!DOCTYPE html>
<html lang = "pt-br">
<head>
   <meta charset = "UTF-8">
   <meta= "viewport" content="width=device-width, initial-scale=1.0">
   <title> Site Simples <title>
</head>
<body>
</body>
</html>

  • 2° Passo:


Neste passo, adicionaremos a tag <style> dentro do <head>, para que possamos, assim, fazer a estilização dos recursos sem a necessidade de um arquivo CSS externo. Ficará desta forma:


<!DOCTYPE html>
<html lang = "pt-br">
<head>
   <meta charset = "UTF-8">
   <meta= "viewport" content="width=device-width, initial-scale=1.0">
   <title> Site Simples <title>
      <style>
	   body { 
                font-family : Arial, sans-serif; = 
                margin: 0;
                padding: 0;
                background-color: "#f4f4f4"; 
                }

	   .container { 
		width: 80%;
		margin: auto;
		overflow: hidden;
		}
									
	   header  { 
		background : 50b3a2; 
	        color: #ffffff;
		padding-top: 30px;
		min-height: 70px;
		border-bottom: #e8491d 3px solid; 
		}
										
	   header a { 
		color : #ffffff; 
		text-decoration: none;
		text-transform: uppercase;
		font-size: 16px;
		}
		
	   header ul { 
		padding : 0; 
		list-style: none;
		}
		
	   header li { 
		display : inline; 
		padding: 0 20px 0 20px;
		}

	   header #branding { 
		float : right; 
		}
		
	   header #branding h1 { 
		margin : 0; 
		}
		
	   header nav  { 
		float : right; 
		margin-top : 10px; 
		}
			
	   header .highlight, header.current a { 
		color : #e8491d; 
		font-weight : bold; 
		}
      </style>
</head>
<body>
</body>
</html>
