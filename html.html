Tudo o que você precisa saber sobre HTML



As tags são os comandos de uma determinada linguagem, em <HTML> todas as tags são escritas entre esses sinais "<" e ">".

A maioria das tags possuem uma tag de abertura e uma de encerramento, as tags de encerramento possuem uma barra antes do comando da tag, como exemplificado a seguir:

<html> <-- tag de abertura;

</html> <-- tag de encerramento.


Quais são as Principais tags de HTML?

  • Comentários ou <!-- -->


Tag que adiciona um comentário. Em um comentário pode-se escrever qualquer coisa, e não será exibido na tela criada.

É comumente usado para explicar a linha de raciocínio do código escrito, para ajudar os outros devs que estiverem trabalhando no projeto. Sendo exibido desta forma:

<!-- Seu comentário aqui -->

  • HTML


<html> é o elemento base da linguagem HTML, ela é a tag principal de qualquer código, todos os outros elementos estarão dentro desta tag.

  • Head


<head> Providência informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.

  • Title


<title> Tag que adiciona um "nome" para a aba do seu website no navegador.

  • Body


A tag <body> dá acesso ao corpo do site, é aqui que as tags de comando são escritas. São essas que serão exibidas no site.

  • h1 à h6


<h1> à <h6> É uma tag de hierarquia de título, em que <h1> possui o maior nível hierárquico, e <h6> possui o menor.

  • p


A tag <p> adiciona um parágrafo.

  • br


<br> Tag responsável pela quebra de linha dentro dos códigos. É uma tag que não necessita de uma tag de encerramento.

  • b


<b> A tag que transforma o texto em negrito.

  • i


<i> Tag que transforma o texto em itálico.

  • u


<u> Tag que acrescenta um sublinhado no que for escrito.

  • Strike


<strike> Tag que risca a palavra. Exemplo: palavra.

  • Sub


<sub> tag que marca fragmentos de texto que devam ser renderizados em alinhamento vertical mais baixo em relação ao texto principal.

  • Sup


<sup> Tag que marca fragmentos de texto que devam ser renderizados em alinhamento vertical mais alto em relação ao texto principal.

Tags especiais no HTML:




  • Listas Ordenadas e não Ordenadas


A principal função das tags é criar uma lista. A diferença entre elas são que, a tag <ul> define uma lista não ordenada. Já a tag <ol> tem como função definir uma lista ordenada.

A lista <ul> tem três tipos de modelos, que definem o ícone de marcação de cada item, como:

circle - círculo não preenchido;

square – quadrado;

disc - círculo preenchido "padrão".

Exemplo: <ul type = " circle ">

A lista <ol> também possui modelos específicos para os ícones de marcação de cada item, mas, por ser uma lista ordenada, são comuns valores numéricos ou sequências lógicas. Como:

I - Algarismo romano maiúsculo;

i - algarismo romano minúsculo;

a - de A à Z minúsculo;

A - de A à Z maiúsculo;

1 - 1 até o número desejado.

<li> É a tag responsável por definir um item na lista.

  • Formulários


O elemento <form> abre um campo para a criação de um formulário. Possui vários elementos para criar-se o formulário, como as seguintes tags:

Text - criará uma caixa de texto.

Password – Não irá exibir a senha, exibindo pontos ou asteriscos para garantir a privacidade da senha.

Button - criará um botão.

Submit - captura de dados. Exemplo: cadastrar-se em um site exige que você escreva todas as suas informações, como: nome, e-mail, senha, tel. e etc. Basicamente ele enviará todos os dados preenchidos para o site.

value - Define o texto para o botão.

  • a


A principal função do elemento <a> é descrever um link, conectando os diversos documentos de um site e permitindo a navegação por esse conteúdo. Normalmente esses documentos estão relacionados por compartilharem um assunto em comum.

  • Imagens e Figuras


O elemento <figure> é uma marcação de uso específico para a inserção de uma figura. Para incluir a descrição dessa figura, podemos utilizar o elemento <figcaption>.

Elemento que insere de fato a imagem dentro do código é o <img>, comumente, dentro da tag <figure>. Usado da seguinte forma:

<figure>
<img src=” alguma_coisa.jpg ”>
</figure>

  • Table


Uma tabela em HTML começa sempre pela tag <table>. A estrutura da tabela fica dentro dessas tags

Tags semânticas no HTML

  • Header


o <header> é utilizado para representar o cabeçalho de um documento ou seção declarado no HTML. Nele podemos inserir elementos de <h1> a <h6>, até elementos para representar imagens, parágrafos ou mesmo listas de navegação.

  • nav


Utiliza-se <nav> quando deseja-se representar um agrupamento de links de navegação, que, por sua vez, são criados com os elementos <ul>,<li> e <a>.

  • em


O elemento <em> é utilizado quando desejamos enfatizar um trecho ou palavra no texto, indicando que ela contribui de forma mais relevante para o sentido/compreensão do conteúdo.

  • Strong


O elemento <strong> também é utilizado para destacar uma parte do texto. Sua principal diferença em relação ao elemento <em> é que <em> pode alterar o propósito de uma frase, como vimos anteriormente.
