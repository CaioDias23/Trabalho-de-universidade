Conhecimentos prévios necessários para a criação de um web site

A criação de um website simples não é algo tão complexo, podendo ser feito por qualquer um que estude o básico de HTML e CSS.

Utilizaremos as linguagens descritas anteriormente, pois, para a criação de um site simples, já são o suficiente. Essas linguagens são escritas em um bloco de notas especial, que é um editor de códigos.

Quais são os principais editores de códigos?

Apresentaremos dois bons editores de código gratuitos, um deles é o NotePad++ e o VSCode.

Recomenda-se o VSCode, por possuir uma interface muito mais completa, vastas extensões que facilitaram seus projetos, além de preenchimento automático e destaque de erros, que reduzem muito seu esforço e aumentam extremamente sua produtividade.

O que você precisa saber sobre HTML e CSS:



O que é HTML?


HTML é linguagem de marcação de hipertexto padrão para criação de páginas da Web, descreve a estrutura de uma página da Web.

Que consiste em uma série de elementos que informam ao navegador como exibir o conteúdo, esses elementos rotulam partes do conteúdo como “este é um título”, “este é um parágrafo”, “este é um link”, etc.

Clique aqui para se aprofundar em html

O que é CSS?

CSS significa Folhas de Estilo em Cascata, ela descreve como os elementos HTML devem ser exibidos na tela, no papel ou em outras mídias.

Tem o papel de economizar muito trabalho já que ele pode controlar o layout de várias páginas da web de uma só vez, além de que Folhas de estilo externas podem ser armazenadas em arquivos CSS.

Por que usar CSS?

CSS é usado para definir estilos para suas páginas da web, incluindo design, layout e variações de exibição para diferentes dispositivos e tamanhos de tela.
